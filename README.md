# Regenera — Acción Socioambiental

Prototipo web one-page para la consultora chilena **Regenera Acción Socioambiental**.

**URL pública:** https://d1egol.github.io/Regenera/

## Stack

- Prototipo actual: HTML + CSS + JS estático (un solo archivo `index.html`), desplegado en GitHub Pages.
- Producción futura: WordPress + Elementor Pro + WPML + Google Workspace.

## Estructura

One-page con 8 secciones: Hero, Propósito, Servicios, Metodología, Equipo, Valores, Clientes y Contacto.

Ver [index.html](index.html) — todo el CSS y JS está inline y el logo va como SVG; las únicas dependencias externas son Google Fonts y fotos de Unsplash.
